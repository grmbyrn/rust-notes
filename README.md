## Recommended Setup

To get the best experience viewing these notes, especially if you use VS Code, we highly recommend installing the **Markdown Preview Enhanced** extension. It provides advanced Markdown rendering, including code highlighting, diagrams, and more.

### How to install Markdown Preview Enhanced in VS Code

1. Open VS Code
2. Go to the Extensions panel (Ctrl+Shift+X or Cmd+Shift+X)
3. Search for `Markdown Preview Enhanced`
4. Click **Install**
